№1  Поиск отелей.++
    1. Переходим на сайт www.ostrovok.ru.
    2. Вводим в поле "Направление" населённый пункт.
    3. Выбираем дату заселения.
    4. Выбираем дату отъезда.
    5. Выбираем количество номеров.
    6. Выбираем количество взрослых и детей(до 18лет).
    7. Выбираем цель остановки в отеле.
    8. Нажимаем кнопку "Найти".
    
    Ожидаемый результат: появится список отелей, хостелов апартаментов, подобранных по нашим параметрам.

№2  Тестирование сортировки по цене+
    1. Переходим на сайт www.ostrovok.ru.
    2. Заполняем данные для поиска жилья.
    3. Нажимаем "Найти".

    Ожидаемый результат: Список жилья по нашим параметрам обновится. Пояявится список, который будет отсортирован по цене начиная с дешёвых.

№5  Бронирование номера.+
    1. Переходим на сайт www.ostrovok.ru.
    2. Заполняем данные для поиска жилья.
    3. Нажимаем "Найти".
    4. Выбираем отель.
    5. Нажимаем кнопку "Просмотреть цены".
    6. Выбираем номер.
    7. Нажимаем кнопку "Забронировать".
    8. Заполняем "Контактные данные".
    9. Заполнить дату и время заезда.
    10. Нажимаем кнопку "Завершить бронирование".

    Ожидаемый результат: появится сообщение о успешном бронировании и будет отправлено письмо на эл.почту с данными.
